4. Definisi dan Cara Kerja Hosting

Hosting Web adalah layanan yang menyediakan penyimpanan dan infrastruktur yang diperlukan agar sebuah situs web atau aplikasi dapat diakses secara online oleh pengguna di berbagai perangkat.

Konsep Server: Setiap website terdiri dari banyak file (gambar, kode, teks) yang harus disimpan di komputer khusus berkemampuan tinggi yang disebut Server. Penyedia layanan hosting bertugas memelihara, mengonfigurasi, dan menjalankan server fisik tersebut.

Cara Kerja Hosting Web:

  1. Alokasi Ruang: Penyedia hosting mengalokasikan ruang disk dan sumber daya (RAM, CPU) di servernya untuk file-file website Anda.
  2. Penyimpanan File: Semua file situs web (kode, gambar, teks) disimpan di server tersebut.
  3. Pengiriman Respons: Ketika Domain Anda diakses, server hosting menerima permintaan dan mengirimkan file-file tersebut melalui internet ke browser pengunjung.
