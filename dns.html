Definisi dan Cara Kerja DNS (Domain Name System)

1. Definisi DNS

DNS (Domain Name System) adalah sistem terdistribusi hierarkis yang bertugas menerjemahkan nama domain yang dapat dibaca manusia (contoh.com) menjadi Alamat IP numerik (142.250.71.100).

  • Fungsi Utama: DNS adalah "Buku Telepon Digital" Internet yang menghubungkan Domain (alamat) dengan Hosting (lokasi fisik Server).

2. Cara Kerja DNS:

Proses pencarian Alamat IP oleh DNS dikenal sebagai DNS Lookup:

  1. Permintaan (DNS Request): Pengguna mengetik domain, dan perangkat mengirimkan permintaan ke DNS Resolver.
  2. Resolusi Nama Domain: DNS resolver mencari alamat IP yang sesuai dengan nama domain tersebut. Jika alamat IP tidak ditemukan di server resolver, maka permintaan akan diteruskan ke Root DNS Server.
  3. Pencarian Hierarkis: Root DNS Server mengarahkan permintaan ke Top-Level Domain (TLD) Server, misalnya server .com atau .org. TLD Server lalu mengarahkan permintaan ke Authoritative DNS Server yang memiliki data lengkap tentang nama domain tersebut.
  4. Mengirim Balasan ke Resolver: Setelah mendapatkan alamat IP dari Authoritative DNS Server, DNS resolver akan mengirimkan alamat IP tersebut kembali ke perangkat pengguna.
  5. Akses Situs: Dengan alamat IP yang didapatkan, perangkat kemudian dapat menghubungi server tujuan dan menampilkan konten situs web yang diinginkan.
