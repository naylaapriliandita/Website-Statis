6. Definisi dan Cara Kerja Browser Web

Peramban Web (Web Browser) adalah perangkat lunak aplikasi (seperti Google Chrome, Mozilla Firefox, atau Safari) yang menjadi antarmuka bagi pengguna untuk menjelajahi World Wide Web (WWW).

Cara Kerja Browser:

  1. Input dan Permintaan IP: User mengetik URL. Browser mengambil Alamat IP dari DNS.
  2. Akses Server: Browser menggunakan IP untuk menghubungi Server Web (Hosting).
  3. Penerimaan Data: Server memberikan data konten (HTML, CSS, dan file lainnya).
  4. Rendering Konten: Browser menggunakan mesin rendering-nya untuk memproses kode dan menampilkan hasil visual yang utuh kepada user.
