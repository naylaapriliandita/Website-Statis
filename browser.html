Definisi dan Cara Kerja Browser Web

1. Definisi Browser

Browser (Web Browser) adalah perangkat lunak (software) yang berfungsi sebagai antarmuka (interface) utama yang digunakan pengguna untuk mengakses, mencari, dan menampilkan informasi dari World Wide Web (WWW).

  • Peran: Browser adalah Client dalam arsitektur Client-Server. Tugas utamanya adalah menerjemahkan kode-kode web yang kompleks (seperti HTML, CSS, dan JavaScript) menjadi tampilan visual yang mudah dilihat di layar Anda.
  • Contoh: Google Chrome, Mozilla Firefox, Microsoft Edge, Safari.

2. Cara Kerja Browser:

Browser adalah titik akhir dari semua proses yang sudah kita bahas (Internet, DNS, HTTP/HTTPS). Ini adalah alur kerjanya secara ringkas:

  1. Input dan Menerjemahkan Alamat: User mengetik URL. Browser memulai proses DNS Lookup untuk mengubah nama domain yang diketik menjadi Alamat IP server tujuan.
  2. Akses Server: Browser menggunakan IP untuk menghubungi Server Web (Hosting). Browser mengirimkan permintaan (request) ke server melalui protokol HTTP atau HTTPS.
  3. Penerimaan Data: Browser menerima paket data respons dari server yang berisi file-file website (HTML, CSS, dan JavaScript). Jika menggunakan HTTPS, browser melakukan dekripsi data terlebih dahulu.
  4. Rendering Konten: Browser menggunakan mesin rendering-nya untuk memproses kode dan menampilkan hasil visual yang utuh kepada user.
    • Membaca kode HTML untuk menentukan struktur halaman.
    • Menerapkan gaya dan tata letak dari kode CSS.
    • Menjalankan kode JavaScript melalui JavaScript Engine untuk membuat elemen interaktif.
  5. Menampilkan Halaman: Browser menampilkan hasil akhir dari proses rendering sebagai halaman web utuh di layar Anda.

Alur kerja browser dari menerima URL hingga menampilkan halaman, termasuk proses DNS, dan Request/Response
